# 六力系统设计文档（简化版 v3）

## 设计原则

1. **无 HP**：命中 → 致命性 + 掷骰 → 伤势
2. **物理输入**：武器参数为可查询的客观数据
3. **三状态基座**：体能、士气、专注，一个口粮滑块驱动前两者
4. **单兵 ↔ 班组**：同样公式，只改变颗粒度注释
5. **对照案例**：德军 Kar98k 步枪手 vs 苏军 M91/30 步枪手

---

## 第〇章：三状态基座

| 状态 | 含义 | 输入 | 影响 |
|------|------|------|------|
| 体能 | 肌肉+代谢+水合的总和 | 口粮 + 训练水平 + 负重 | 移速、消耗率 |
| 士气 | 凝聚力+求生本能的总和 | 战斗经验 + 伤亡冲击 + 口粮 | 崩溃阈值、命令服从 |
| 专注 | 认知带宽 | 战斗经验 + 被压制程度 | 精度、发现概率 |

三个值均为 0~100，不含子变量。

### 口粮滑块

```
口粮: 800~4500 kcal/天, 一个滑块

体能恢复 = (口粮/3500) × 基准恢复率, 上限 1.2
士气恢复 = (口粮/3500) × 基准恢复率, 上限 1.0

口粮 < 2500: 体能每日 −(2500-口粮)/200 × 5
口粮 < 1800: 士气每日 −(1800-口粮)/200 × 3
口粮 < 1000: 求生本能触发概率 +30%
```

| 口粮 | 典型场景 | 体能恢复 | 士气恢复 | 衰减 |
|------|---------|---------|---------|------|
| 4200 | 美军西欧 1944 | ×1.2 | ×1.0 | 无 |
| 3500 | 德军明斯克 1941 | ×1.0 | ×1.0 | 无 |
| 2500 | 德军北非 1942 | ×0.71 | ×0.71 | 体能临界 |
| 1600 | 德军莫斯科 1941.12 | ×0.46 | ×0.46 | 体能−4.5/天, 士气−1.5/天 |
| 1200 | 列宁格勒被围 | ×0.34 | ×0.34 | 体能−6.5/天, 士气−4.5/天, 求生+30% |
| 800 | 绝境 | ×0.23 | ×0.23 | 体能−8.5/天, 士气−7.5/天, 7天后必崩 |

### 状态联动

```
体能 < 30% → 士气消耗 ×1.5（饿兵更容易崩溃）
士气 < 20% → 专注上限 ×0.5（恐惧压倒注意力）
士气归零 → 求生本能接管，拒绝攻击命令
```

### 扩展到班组

| 状态 | 单人 | 班组 |
|------|------|------|
| 体能 | 这个兵的体力 | 全班平均体能，取最差者限制 |
| 士气 | 这个兵的意志 | 全班凝聚力，伤亡整班扣减 |
| 专注 | 这个兵的注意力 | 全班警戒水平，有观察手+5，有机枪手+3 |

## 第〇章补：武器物理推导

### 输入

| 参数 | 类型 | Kar98k | M91/30 |
|------|------|--------|--------|
| `caliber` | mm | 7.92 | 7.62 |
| `action` | bolt/semi/auto | bolt | bolt |
| `barrelLength` | mm | 600 | 730 |
| `sightMag` | 倍率 | 1.5 | 1.0 |

### 推导 1：精度

```
精度 = 0.4 + barrelLength/2000 + (sightMag - 1) × 0.05

Kar98k:  0.4 + 600/2000 + 0.025 = 0.4 + 0.30 + 0.025 = 0.725
M91/30:  0.4 + 730/2000 + 0     = 0.4 + 0.365 + 0     = 0.765
```

| 项 | 含义 |
|-----|------|
| 0.4 | 完成基础射击训练者在理想条件下的命中基线 |
| barrelLength/2000 | 瞄准基线越长 → 照门准星间距越大 → 角误差越小 |
| (sightMag-1)×0.05 | 倍率对目标辨识的加成 |

### 推导 2：射程 (m)

```
有效射程 = caliber × barrelLength / 7

Kar98k:  7.92 × 600 / 7 = 679 m
M91/30:  7.62 × 730 / 7 = 795 m
StG44:   7.92 × 419 / 7 = 474 m
MP40:    9.0 × 251 / 7 = 323 m
M1911:   11.43 × 127 / 7 = 207 m
```

除 7 将 Kar98k 校准到约 680m（机瞄有效射程的合理值）。

### 推导 3：射速

```
RPM: bolt→15, semi→40, auto→120
射击间隔(ms) = 60000 / RPM

Kar98k:  60000 / 15 = 4000 ms
M91/30:  60000 / 15 = 4000 ms
```

### 推导 4：致命性

```
致命性 = caliber × actionKillMod

actionKillMod: 口径本身的打击力修正，暂定 1.0（后续可调整）
```

| 口径 | 致命性 | 命中后果 |
|------|--------|---------|
| 7.92mm (Kar98k) | 8 | 极高致命 |
| 7.62mm (Mosin) | 8 | 极高致命 |
| 9mm (MP40) | 5 | 中等致命 |
| 11.43mm (.45 ACP) | 6 | 中高致命（大口径手枪弹） |
| 12.7mm (.50 BMG) | 13 | 过度杀伤 |

**致命性判定（替代 WL + 能量代理 + 阈值）：**

```
掷 1d10:
  d10 ≤ 致命性 − 4  → 致命
  d10 ≤ 致命性      → 重伤
  其余               → 轻伤
```

| 武器 | 致命性 | 致命区间 | 重伤区间 | 轻伤区间 |
|------|--------|---------|---------|---------|
| Kar98k | 8 | d10≤4 (40%) | d10≤8 (40%) | d10=9,10 (20%) |
| MP40 | 5 | d10≤1 (10%) | d10≤5 (40%) | d10=6~10 (50%) |
| M1911 | 6 | d10≤2 (20%) | d10≤6 (40%) | d10=7~10 (40%) |
| .50 BMG | 13 | d10≤9 (60%*) | d10=10 (10%*) | — |

* 致命性超过 10 后：致命 = d10≤min(致命性−4, 9)，余下全为重伤。d10=10 永远为重伤（免除即死，留 10% 运气）。

### 距离衰减

```
ratio = distance / effectiveRange

ratio ≤ 0.5:       命中修正 = 1.00
0.5 < ratio ≤ 1.0: 命中修正 = 1.00 − (ratio−0.5) × 0.40
ratio > 1.0:       命中修正 = max(0, 0.80 − (ratio−1.0) × 1.60)
```

## 第一章：打击力

### 输入

| 变量 | 来源 | 德军 | 苏军 |
|------|------|------|------|
| 武器精度 | 推导1 | 0.725 | 0.765 |
| 武器射程 | 推导2 | 679 m | 795 m |
| 武器射速 | 推导3 | 4000 ms | 4000 ms |
| 致命性 | 推导4 | 8 | 8 |
| 专注(当前) | 三状态 | 70 | 50 |
| 训练·掩体射击 | 卡牌(+0.10) | ✓ | ✗ |
| 训练·快速换弹 | 卡牌(ROF×0.9) | ✓ | ✗ |

### 过程

```
实际精度 = 武器精度 × (1 + 训练修正) × 专注修正
          专注: >60→×1.0, 30~60→×0.85, <30→×0.65

德军: 0.725 × 1.10 × 1.00 = 0.798
苏军: 0.765 × 1.00 × 0.85 = 0.650

命中率 = 实际精度 × 距离衰减 × 目标掩体系数 × 射手伤势/压制修正

德军 400m vs 苏军(avg掩体):
  距离衰减: ratio=400/679=0.589 → 1.0−0.036=0.964
  命中率 = 0.798 × 0.964 × 0.50 = 0.385 → 38.5%

射速: 德军 4000×0.9=3600ms, 苏军 4000ms

命中后: 掷 1d10 vs 致命性8
  致命(40%), 重伤(40%), 轻伤(20%)
```

### 输出

| | 德军 | 苏军 |
|------|------|------|
| 实际精度 | **0.798** | 0.650 |
| 实际射速 | **3600ms** | 4000ms |
| 致命/重伤/轻伤 | 40/40/20% | 40/40/20%(同枪) |
| 实战命中率(400m) | **38.5%** | 26.1% |

**专注力差距使苏军精度打八五折。但双方 Kar98k 和 Mosin 的致命性相同(口径接近)，命中后的结果等效。**

## 第二章：生存力

### 输入

| 变量 | 类型 | 德军 | 苏军 |
|------|------|------|------|
| 护甲 | none/light/heavy | none | none |
| 掩体利用 | poor/avg/good | good | avg |
| 士气(当前) | 三状态 | 75 | 30 |
| 战斗经验 | 新兵/老兵/精锐 | 老兵 | 新兵 |

### 过程

**护甲（吸收致命性——不是减伤，是降低致命区间）：**

```
护甲等级 → 降低攻击方致命性:
  none:  不变
  light: 致命性 −1（降低手枪/冲锋枪致命率，不影响步枪）
  heavy: 致命性 −2

Kar98k(致命8) vs 无甲: 致命 40%, 重伤 40%
Kar98k(致命8) vs 轻甲: 致命8−1=7, 致命30%, 重伤50%
Kar98k(致命8) vs 重甲: 致命8−2=6, 致命20%, 重伤60%
// 护甲没法把步枪弹变成轻伤——但它降低了致命概率
```

**掩体利用（降低被命中概率，不是减伤）：**

```
敌方命中率 × 掩体系数:
  good: ×0.30, avg: ×0.50, poor: ×0.75
```

**伤势状态机：**

| 伤势 | 移速 | 精度 | 开火 | 持续 | 叠加 |
|------|------|------|------|------|------|
| 轻伤 | ×0.5 | ×0.7 | ✓ | 5s | 3轻→重伤 |
| 重伤 | ×0.3 | ×0.4 | 禁3s | 15s→轻伤 | 2重→致命 |
| 致命 | 0 | 0 | ✗ | 永久 | =死 |

**士气在生存中的作用：**

```
受伤对士气的影响:
  轻伤: −5
  重伤: −20
  50m内友军阵亡: −15/人

士气 < 30: 命令延迟 ×1.5
士气 < 20: 自动向最近掩体移动
士气归零: 求生本能接管 → 拒绝攻击 + 仅可自卫 + 自动后撤
士气恢复 > 20: 解除生存模式

恐慌传染: 50m内友军士气 −10（连锁触发）
```

### 输出

| | 德军 | 苏军 |
|------|------|------|
| 护甲 | none(致命8) | none(致命8) |
| 掩体利用 | **good(×0.30)** | avg(×0.50) |
| 士气基线 | **75** | 30 |
| 一战重伤后 | 士气55(可控) | 士气10(濒溃) |
| 一战重伤+目睹一死 | 士气40(谨慎) | 士气−5(崩溃→求生接管) |

## 第三章：机动力

### 输入

| 变量 | 类型 | 德军 | 苏军 |
|------|------|------|------|
| 体能(当前) | 三状态 | 70 | 60 |
| 口粮 | kcal | 3500 | 2800 |
| 总负重 | 推导 | 15.2 kg | 13.6 kg |
| 训练·负重行军 | 卡牌 | ✓(×0.8) | ✗ |

### 过程

```
武器重量 = actionBaseWt + caliber×barrelLength/4000
  actionBaseWt: bolt=3.2, semi=3.5, auto=3.8, pistol=1.0

德军: 3.2 + 7.92×600/4000 = 4.39 kg
苏军: 3.2 + 7.62×730/4000 = 4.59 kg

总负重 = 武器 + 弹药 + 装备6kg + 护甲
德军: 4.39+4.8+6 = 15.2kg, 苏军: 4.59+3.0+6 = 13.6kg

体能消耗 = 负重/10 × 动作系数 × 训练修正
  动作: 步行1.0, 奔跑2.5, 匍匐1.8
德军步行: 1.52 × 0.8 = 1.22/s
苏军步行: 1.36 × 1.0 = 1.36/s

体能恢复(静止) = 5 × 口粮/3500 /s
德军: 5 × 1.0 = 5/s
苏军: 5 × 0.8 = 4/s

移速修正:
  体能 > 60%: ×1.0, 30~60%: ×0.75, 10~30%: ×0.5, <10%: ×0.25
  伤势叠加: 轻伤×0.5, 重伤×0.3
```

### 输出

| | 德军 | 苏军 |
|------|------|------|
| 总负重 | 15.2 kg | 13.6 kg |
| 步行消耗 | **1.22/s** | 1.36/s |
| 体能恢复(静止) | **5.0/s** | 4.0/s |
| 体能基线 | **70** | 60 |

## 第四章：感知力

### 输入

| 变量 | 类型 | 德军 | 苏军 |
|------|------|------|------|
| visionRange | m | 350 | 330 |
| hearingRange | m | 600 | 550 |
| detection | 0~1 | 0.60 | 0.45 |
| sightMag | 倍率 | 1.5 | 1.0 |
| 专注(当前) | 三状态 | 70 | 50 |
| 训练·小队协同 | 卡牌 | ✓(comm×1.3) | ✗ |

### 过程

```
视野 = visionRange × 专注修正(>60×1.0, 30~60×0.85)
德军: 350×1.0=350m, 苏军: 330×0.85=280m

瞄具加成 = (sightMag−1)×0.4 → 德军 +0.20

发现率 = (detection+瞄具加成) × (1−目标暴露) × 地形系数
德军打步行苏军: (0.60+0.20)×0.3 = 24%/0.5s → 平均 2.1s
苏军打步行德军: 0.45×0.3 = 13.5%/0.5s → 平均 3.7s

听觉: 开火→hearingRange×2, 奔跑→×0.7, 步行→×0.3

通讯: voice=200m, 小队协同→×1.3+relay连锁
德军: 260m relay, 苏军: 200m 无relay
```

### 输出

| | 德军 | 苏军 |
|------|------|------|
| 视野 | 350m, FOV=70° | 280m, FOV=90° |
| 发现(vs步行) | **24%/0.5s** | 13.5%/0.5s |
| 平均发现时间 | **2.1s** | 3.7s |
| 通讯 | **260m+relay** | 200m 无relay |

| 变量 | 类型 | 德军 | 苏军 |
|------|------|------|------|
| 纪律 | low/avg/high/elite | high | avg |
| 自主性 | low/med/high | high | low |
| 士气(当前) | 三状态 | 75 | 30 |

### 过程

```
命令延迟 = 纪律延迟 + 士气修正
  纪律: elite=0, high=500ms, avg=1500ms, low=3000ms
  士气: >60→×1.0, 30~60→×1.2, <30→×1.5

德军: 500ms, 苏军: 1500×1.3=1950ms

恐慌判定(重伤/目睹阵亡/士气<30被射击):
  纪律 d100: elite≤5(5%), high≤15(15%), avg≤30(30%), low≤50(50%)
  失败→断裂10~40s，50m内士气−10(传染)

自主模式(失联后):
  high→继续命令+找掩体+还击+靠拢友军
  med→继续命令+还击
  low→停止+仅100m还击+可能后撤
```

| | 德军 | 苏军 |

## 附录：公式速查

```
武器:
  精度 = 0.4 + barrelLength/2000 + (sightMag-1)×0.05
  射程 = caliber × barrelLength / 7
  射速 = 60000 / actionRPM
  致命性 = caliber × actionKillMod
  命中后掷 1d10: ≤(致命性-4)=致命, ≤致命性=重伤, 其余=轻伤

打击力:
  实际精度 = 武器精度 × (1+训练修正) × 专注修正
  距离衰减: ratio = dist/range, ≤0.5→1.0, 0.5~1.0→1.0-(r-0.5)×0.4, >1.0→max(0,0.8-(r-1.0)×1.6)
  命中率 = 实际精度 × 距离衰减 × 掩体系数 × 伤势修正 × 压制修正
  实速 = 射击间隔 × (1-换弹训练修正)

生存力:
  护甲: none=致命性不变, light=-1, heavy=-2
  掩体: good=×0.30, avg=×0.50, poor=×0.75
  伤势叠加: 3轻→重伤, 2重→致命
```
