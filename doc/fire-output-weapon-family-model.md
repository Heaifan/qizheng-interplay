# 全陆军武器 FireOutput 火力输出模型

> 本文档定义《奇正相生》全陆军武器火力输出的统一框架。
> 流程统一，公式分族，运行时查表。

---

## 一、总框架

所有陆军武器都按这条链处理：

```
武器原始参数
    ↓
武器族核心系数生成器
    ↓
目标基础输出表（TargetBase）
    ↓
距离修正 × 防护修正
    ↓
最终火力输出
```

最终公式：

```
FireOutput = TargetBase × RangeFactor × ProtectionFactor
```

## 二、武器族分类（8族）

| # | 武器族 | 代表武器 | 核心参数 | 主要目标 |

| 3 | 火炮/迫击炮 | 迫击炮、榴弹炮、火箭炮 | 弹重、装药、破片、半径 | 人员、工事 |

## 三、各武器族生成器

### 3.1 枪械动能类

核心系数：
```
单发毁伤系数 = f(弹丸质量, 初速, 口径)
穿透系数     = f(弹头结构, 口径, 初速)
结构破坏系数 = f(弹丸质量, 口径)
```

目标基础输出：
```
人员     = 单发毁伤系数
轻车辆   = 穿透系数 × 0.70 + 结构破坏系数 × 0.20
装甲     = 穿透系数 × 0.04
工事     = min(结构破坏系数 + 穿透系数 × 0.04, 枪械工事上限)
障碍     = 结构破坏系数
```

适用：手枪、冲锋枪、栓动步枪、半自动步枪、突击步枪、轻机枪、通用机枪。

> MG42 和 Kar98k 用同一弹种时，单发 FireOutput 接近是合理的。
> MG42 的优势体现在压制能力、火力持续、火力密度，不在单发毁伤。

---

### 3.2 重机枪/反器材动能类

核心系数：
```
单发毁伤系数 = f(弹丸质量, 初速, 口径)
穿透系数     = f(弹头结构, 口径, 穿甲设计等级)
结构破坏系数 = f(弹丸质量, 口径) × 1.2
反器材系数   = f(穿甲设计等级, 口径)
```

目标基础输出：
```
人员     = 单发毁伤系数 × 1.10
轻车辆   = 穿透系数 × 0.85 + 结构破坏系数 × 0.30
装甲     = 穿透系数 × 装甲适配系数
工事     = 结构破坏系数 × 0.60 + 穿透系数 × 0.20
障碍     = 结构破坏系数 × 0.80
```

适用：12.7mm/14.5mm 重机枪、反坦克枪、反器材步枪。

---

### 3.3 火炮/迫击炮爆炸破片类

核心系数：
```
爆炸毁伤系数 = f(装药量, 口径)
破片杀伤系数 = f(弹体重, 破片设计)
覆盖系数     = f(爆炸半径)
结构破坏系数 = f(装药量, 弹体重量)
```

目标基础输出：
```
人员     = 破片杀伤系数 × 0.60 + 爆炸毁伤系数 × 0.25 + 覆盖系数 × 0.15
轻车辆   = 爆炸毁伤系数 × 0.35 + 破片杀伤系数 × 0.25 + 结构破坏系数 × 0.20
装甲     = 爆炸毁伤系数 × 0.08 + 结构破坏系数 × 0.05
工事     = 爆炸毁伤系数 × 0.40 + 结构破坏系数 × 0.50
障碍     = 爆炸毁伤系数 × 0.30 + 结构破坏系数 × 0.60
```

适用：迫击炮、步兵炮、榴弹炮、野战炮、火箭炮、坦克榴弹。

---

### 3.4 手榴弹/榴弹/榴弹发射器类

核心系数：
```
近距爆炸系数   = f(装药量)
破片杀伤系数   = f(弹体重, 破片设计)
小范围覆盖系数 = f(爆炸半径)
结构破坏系数   = f(装药量)
```

目标基础输出：
```
人员     = 破片杀伤系数 × 0.65 + 近距爆炸系数 × 0.25 + 小范围覆盖系数 × 0.10
轻车辆   = 近距爆炸系数 × 0.20 + 破片杀伤系数 × 0.15
装甲     ≈ 0（除非是破甲榴弹 → 走聚能破甲类）
工事     = 近距爆炸系数 × 0.25 + 结构破坏系数 × 0.25
障碍     = 结构破坏系数 × 0.20
```

适用：手榴弹、枪榴弹、榴弹发射器、掷弹筒。

---

### 3.5 反坦克炮/坦克炮穿甲类

核心系数：
```
穿透系数         = f(穿深等级, 弹种)
装甲后效系数     = f(命中后效, 弹种)
结构破坏系数     = f(弹丸质量, 初速)
人员附带毁伤系数 = f(弹种, 口径)
```

目标基础输出（穿甲弹）：
```
人员     = 人员附带毁伤系数
轻车辆   = 穿透系数 × 0.70 + 结构破坏系数 × 0.40
装甲     = 穿透系数 × 装甲后效系数
工事     = 结构破坏系数 × 0.40 + 穿透系数 × 0.20
障碍     = 结构破坏系数 × 0.30
```

注意坦克炮可按弹种绑定多个输出档案：
- 穿甲弹：装甲输出高
- 高爆弹：人员/工事输出高
- 破甲弹：装甲与工事都有一定输出

适用：反坦克炮、坦克炮、机关炮（穿甲弹种）。

---

### 3.6 火箭筒/聚能破甲类

核心系数：
```
聚能穿透系数   = f(战斗部直径, 标称穿深)
装甲后效系数   = f(装药结构)
爆炸附带系数   = f(装药量)
结构破坏系数   = f(战斗部直径)
```

目标基础输出：
```
人员     = 爆炸附带系数 × 0.40 + 破片附带系数 × 0.20
轻车辆   = 聚能穿透系数 × 0.70 + 爆炸附带系数 × 0.20
装甲     = 聚能穿透系数 × 装甲后效系数
工事     = 爆炸附带系数 × 0.30 + 结构破坏系数 × 0.40
障碍     = 结构破坏系数 × 0.30
```

适用：火箭筒、无后坐力炮（破甲弹）、聚能破甲弹、反坦克榴弹。

---

### 3.7 火焰/燃烧类

核心系数：
```
燃烧毁伤系数   = f(燃烧强度)
覆盖系数       = f(覆盖面积)
持续作用系数   = f(持续时间, 附着能力)
工事清除系数   = f(封闭空间效应)
```

目标基础输出：
```
人员     = 燃烧毁伤系数 × 0.50 + 覆盖系数 × 0.30 + 持续作用系数 × 0.20
轻车辆   = 燃烧毁伤系数 × 0.30 + 持续作用系数 × 0.20
装甲     = 燃烧毁伤系数 × 0.10（特殊燃烧效应另计）
工事     = 工事清除系数 × 0.70 + 持续作用系数 × 0.30
障碍     = 燃烧毁伤系数 × 0.20
```

适用：火焰喷射器、燃烧瓶、白磷弹、燃烧弹。

火焰武器对工事内人员有额外加成（通过封闭空间修正实现）。

---

### 3.8 爆破/工兵破障类

核心系数：
```
爆破毁伤系数   = f(装药量, 接触方式)
结构破坏系数   = f(装药量, 爆破类型)
清障系数       = f(破障用途)
人员附带杀伤系数 = f(装药量)
```

目标基础输出：
```
人员     = 人员附带杀伤系数
轻车辆   = 爆破毁伤系数 × 0.40
装甲     = 爆破毁伤系数 × 0.20（除非是专门反坦克装药）
工事     = 结构破坏系数 × 0.80 + 爆破毁伤系数 × 0.30
障碍     = 清障系数 × 1.00
```

适用：炸药包、爆破筒、爆破索、工兵装药。

## 四、运行时查表

战斗中不实时计算武器族公式。所有武器族生成器只在开发/数据生成阶段运行一次，产物是**武器输出档案**：

```
WeaponOutputProfile {
  id:        string
  label:     string
  targetBase: {
    personnel,
    lightVehicle,
    armor,
    structure,
    obstacle
  }
  rangeModelId: string
  description:  string
}
```

运行时只需：

```
resolveWeaponOutputProfile(weapon)  →  WeaponOutputProfile
getRangeFactor(rangeM, rangeModelId) →  rangeFactor + rangeBand
PROTECTION_FACTORS[protectionLevel]  →  protectionFactor

FireOutput = targetBase × rangeFactor × protectionFactor
```

## 五、设计原则

1. **流程统一** — 所有武器走同一套计算链
2. **公式分族** — 每个武器族有自己的核心系数生成器
3. **运行时查表** — 战斗中不实时算物理，只读已生成的输出档案
4. **族间差异在系数，不在公式结构** — 所有武器输出流程一致，差异体现在靶标分配权重上
